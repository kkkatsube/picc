# Frontend (React SPA)

React 18 + TypeScript + Vite による SPA

## 技術スタック
- React 18 + TypeScript
- Vite (ビルドツール)
- Tailwind CSS + Heroicons
- React Query + Zustand
- Axios (HTTP Client)

## セットアップ
```bash
# React + TypeScript環境構築
npm create vite@latest . -- --template react-ts
npm install

# 開発サーバー起動
npm run dev
```
